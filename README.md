## Description

Post test app

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Update environment
Clone file ```.env.template``` and rename ```.evn```

## Up db
```
docker-compose up -d
```
